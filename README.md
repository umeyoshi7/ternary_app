# LLE/VLE calculator

3成分系液液平衡（LLE）および気液平衡（VLE）をブラウザ上で計算・可視化する Streamlit アプリケーションです。

## 起動方法

```bash
pip install streamlit thermo plotly pandas
streamlit run app.py
```

---

## 機能一覧

### タブ 1 — LLE 線図

Water を固定した 3成分系（Water + 水溶性溶媒 + 非水溶性溶媒）の液液平衡を計算します。

- **三角図（直角三角座標）** にタイライン・バイノーダル点を表示
- **UNIFAC Dortmund モデル**（活量係数）による相平衡計算
- 温度スライダー（10〜100°C）・格子点数スライダー（10〜50）で条件を変更
- 溶媒・温度・格子点数のいずれかが変更されると**自動再計算**
- タイラインデータを **CSV ダウンロード**可能

**仕込み組成 → 層分離計算**（タブ内左カラム）

- Water / 水溶性溶媒 / 非水溶性溶媒の量を g / mol / mL 単位で入力
- 計算結果を **mol% / w/w% / v/v%** の表形式で水層・有機層それぞれ表示
- 相比率（β値）、三角図上への仕込みタイライン描画

---

### タブ 2 — 蒸気圧曲線

純成分の蒸気圧 vs 温度曲線を表示します。

- 対象成分：全 63成分（Water + 水溶性 25種 + 非水溶性 37種）から選択
- 温度範囲：-50〜250°C のスライダーで自由設定
- **大気圧（101.325 kPa）での沸点**を自動計算・表示
- 101.325 kPa の水平基準線を重ねて表示
- 表示範囲が蒸気圧相関式の有効範囲外の場合は**警告を表示**

---

### タブ 3 — VLE 線図（2成分系）

任意の 2成分系の気液平衡を計算し、2種の線図を表示します。

| 線図 | 内容 |
|------|------|
| xy 線図 | 液相モル分率（x₁）vs 気相モル分率（y₁）、対角線付き |
| T-xy 線図 | モル分率 vs 泡点温度（青）・露点温度（橙破線）・三相水平線（緑破線） |

- 圧力を kPa 単位で任意指定
- 共沸点の検出は計算結果から読み取り可能
- 不均一共沸系（LL 分離先行系）も泡点二分探索により計算可能
- **異質共沸系（Water–n-Hexane 等）の三相表示**：
  - T-xy 線図に三相温度 T₃ の水平破線（緑）を表示
  - α端点・β端点（液液飽和組成）を緑丸マーカーで明示
  - 異質共沸気相組成 y₃ を赤ダイヤモンドマーカーで表示
  - 三相域内の泡点・露点を相律に従い T₃ で補完（欠損・誤収束を修正）

### タブ 4 — 濃縮シミュレーション（レイリー蒸留）

混合物を連続的に蒸発させた場合の組成・沸点変化をシミュレーションします。

- **2〜4 成分**を全 63 成分から自由選択
- 入力単位：mol / g / mL（mL 使用時は仕込み温度を指定して密度補正）
- 圧力を kPa 単位で任意指定
- 計算結果をグラフ表示：
  - 第1縦軸：各成分量 + 合計（入力単位）vs 蒸発割合（0→1）
  - 第2縦軸：沸点（°C）vs 蒸発割合

**アルゴリズム：微分蒸留（Rayleigh 方程式）**

各ステップで泡点フラッシュを実行し、平衡蒸気組成 y を求めて液相組成を更新します。

```
z = L / sum(L)
res = flash(P, VF=0, z)   → 泡点温度 T_bp、平衡蒸気組成 y
L_i ← L_i − ΔV × y_i
```

不均一系（LL 分離先行）では VF=0 仕様が失敗するため、T-P 二分探索にフォールバック。
三相共存域（T・y が一定）はキャッシュ再利用によって高速化しています。

**三相プラトー検出**：沸点が 0.5°C 以内で 3ステップ以上連続した場合、三相共存域（Gibbs 相律により T・y が一定）として画面上に案内メッセージを表示します。

---

### タブ 5 — ロジック

計算に使用している数式・アルゴリズムの解説を掲載しています。

- UNIFAC Dortmund モデルの数式（Combinatorial / Residual 項、温度依存 Ψ）
- LLE 等フガシティー条件・物質収支・Rachford-Rice 方程式
- フラッシュアルゴリズム（TPD 安定性テスト → 逐次代入法）
- 密度計算・単位変換の詳細

## 対応溶媒（63成分）

| 種別 | 成分数 | 例 |
|------|--------|----|
| Water | 1 | Water |
| 水溶性（Miscible） | 25 | Ethanol, Methanol, Acetone, Acetonitrile, IPA, THF, DMF, DMSO, NMP, ... |
| 非水溶性（Immiscible） | 37 | Toluene, n-Hexane, EtOAc, DCM, Chloroform, n-Heptane, Cyclohexane, 2-MeTHF, CPME, ... |

---

## 技術仕様

| 項目 | 内容 |
|------|------|
| 活量係数モデル | UNIFAC Dortmund（thermo ライブラリ） |
| 状態方程式（気相） | Peng-Robinson（PRMIX） |
| 液相フガシティー | GibbsExcessLiquid（蒸気圧相関式 + 活量係数） |
| フラッシャー | FlashVLN（多相多成分） |
| 水密度 | Kell (1975) 多項式（10〜100°C） |
| 有機溶媒密度 | 線形近似 ρ = a + b·T |
| 蒸気圧サロゲート補正 | `vp_T_offset` (K) による温度シフトで実測沸点に合わせる（4-MeTHP 等） |
| UNIFAC 手動割当 | thermo DB 未登録・誤認識化合物（DMI, Nitromethane, TEA）を手動グループ指定 |
